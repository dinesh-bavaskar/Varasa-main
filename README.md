## Overview

VĀRASĀ is a production-ready full stack web application designed to preserve, manage, and showcase cultural heritage initiatives.

The platform combines a responsive public interface with a secure administrative system powered by a RESTful API and JWT-based authentication.

Built with modern full stack architecture and deployed to cloud infrastructure.

---

## Core Capabilities

- Secure Admin Authentication (JWT)
- Protected API Endpoints
- Content Management System (CMS)
- Production-ready REST API
- Environment-based configuration
- Cross-origin resource security (CORS)
- Cloud deployment (Vercel + Render)
